# URL Shortener - Spring Boot

A backend URL shortening service built using Spring Boot and Java.

This application allows users to generate compact short URLs from long URLs and redirect users back to the original destination efficiently.

---

# Features

- Generate shortened URLs
- Redirect shortened URLs to original links
- RESTful API architecture
- Layered backend design
- Database persistence
- Scalable backend structure

---

# Tech Stack

- Java
- Spring Boot
- Spring Web
- Spring Data JPA
- Gradle
- MySQL
